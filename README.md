# COMP 372 - Artificial Intelligence (In-Class Coding Demos)

## Repository Structure

Each branch in this repository contains the code and exercises for a specific class session. Branch names follow the date format `YYYY.MM.DD`. To access a particular day's materials, switch to that branch first:

```bash
git checkout 2026.04.14
```

To return to the main branch:

```bash
git checkout main
```
